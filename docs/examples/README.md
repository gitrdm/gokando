## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/gitrdm/gokando.git
   cd gokando
   ```

2. **Navigate to examples:**

   ```bash
   cd examples
   ```

3. **Run an example:**
   ```bash
   go run example-name/main.go
   ```
