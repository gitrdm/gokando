# API Reference

Complete API documentation for gokanlogic packages.

## Core Packages

### miniKanren & FD Solver

The main package combining relational logic programming (miniKanren) with finite domain (FD) constraint solving.

**[Full API Documentation →](minikanren.md)**

**Relational (miniKanren):** Logic programming with unification
- Query execution, logical relations, list operations

**FD Constraint Solver:** Numeric constraint satisfaction
- Domain variables, global constraints, optimization

**Hybrid Solving:** Combined relational + FD reasoning

### Nominal Logic

Support for alpha-equivalence, name binding, and fresh name generation.

**[Full API Documentation →](nominal.md)**

### Parallel Search

High-performance parallel search execution.

**[Full API Documentation →](parallel.md)**
